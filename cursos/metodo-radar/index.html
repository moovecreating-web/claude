Perguntas frequentes

Preciso saber programar?

Não. O curso foi feito exatamente para quem não sabe programar. Você usa o Claude Code — uma IA que escreve o código por você. Você foca em dizer o que quer, a IA resolve o resto.

Precisa instalar algum software pago?

VS Code e GitHub são 100% gratuitos. O Claude Code tem plano gratuito — se você quer criar o dashboard só para a sua própria conta, ele já resolve. Mas se pretende atender clientes e criar vários dashboards, o plano Pro ($20/mês) é o recomendado — você já recupera no primeiro cliente.

Posso personalizar o dashboard para cada cliente?

Sim — e é exatamente isso o diferencial do Método RADAR. Você escolhe quais métricas exibir (CPL, ROAS, leads, verba, CTR, criativos), as cores da marca, o logo. Cada cliente tem um dashboard único com o link dele.

Funciona para qualquer nicho de cliente?

Sim. O dashboard se conecta direto à API do Meta Ads — funciona para qualquer conta de anúncio, qualquer nicho, qualquer país.
